# Running the tests

## Install criterion and gcovr

### Criterion
On mac:
```brew install snaipe/soft/criterion```

Other installation methods: https://github.com/Snaipe/Criterion

### gcovr:

`pip install gcovr`

## Run the tests

`make run_tests`
